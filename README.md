WAC
===

WeltEnSTurm's Addons Compilation for Garry's Mod.
Contains:
* Aircraft
* Damage System
* Huge Construct Map
* Gravity Controller
* Weapons
* Several Scripts

Installation
------------

Download the .zip file and put the folders inside into your garrysmod/garrysmod/addons folder.
